## Description
These routes allows authenticated users fetch property details.
## Contents

- [Get Properties](#get-properties)
- [Get Single Property](#get-single-property)
- [Get Property Compliance](#get-property-compliance)

## Base URL
http://localhost:5000/api/properties


## Authentication
All endpoints require authentication via a JWT token. See <a href='../authentication.md'>Authentication</a> for details.
