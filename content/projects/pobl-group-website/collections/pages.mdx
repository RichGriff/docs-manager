The first thing you will notice is a list of all Pages that have been created. You can use this screen to:

- Search - select the magnifier icon and type in the page title
- Filter - select the filters button to apply a filter to your list of pages
- Import / Export - allow you to get data in or out of the cms
- Switch language - you can select the language selector, to switch your view to content available in other languages
- Create new entry - allows you to create a new page

#### Creating a new page
You can create a new page by selecting the 'Create new entry' button in the top right of the screen. 

<img src='https://files.gitbook.com/v0/b/gitbook-x-prod.appspot.com/o/spaces%2Fqq1gvL7j0PwJkFX9MJae%2Fuploads%2FU36MgZK6Bb9WUBH8vyZ2%2Fpage-new-cms.JPG?alt=media&token=86f081dd-116b-49a8-b574-de4e7c4f397b' />

The first section here will contain some essential fields for a page:

- Title - is a key field required for navigation
- Display Title - is used for the search functionality
- Description - to provide additional information about this page
- Image - featured image for the page 
- Keywords - enter multiple keywords here separated by ',' - this is also used by the search functionality
- Parent - allows you to select a parent page to this page
- Slug and Full Slug - these are automatic fields, that are populated when you save the page. It uses the title and parent values to create a partial url for the page. 

On the right hand side you have the metadata for the page, with a few extra options. 
- Import Export - allows you to pull or put in data into the record
- Locales - allows you to switch to other language versions.

<Callout>
Locales - currently only Welsh is supported

<b>Title</b>, <b>Description</b> and <b>Image</b> will be visible when you are sharing a link to the page, so these can be very useful.

#### Saving / Publishing
When your ready with your new page, you can select Save in the top right. If your collection doesnt require to be published then this will be instantly available. However, if it requires to the Published then this will first save as a draft. You will then need to select Publish to finalize your new page.

If you require a collection to be updated, whether this is to remove or add the publishing. Please submit a change request.
