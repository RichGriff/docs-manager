#### Fields:
- Heading (required) -  this value determines the slug
- Display Title (required) - the main title shown on the website
- Post Type - select which post group this post belongs to, if nothing selected here, the post will be categorised as General

<Callout>
If there is no types which fits the post, please go to Post Types to create a new one before continuing. 
</Callout>

- Slug - the unique identifier for url

<Callout>
Please don't enter / edit slug manually under any circumstances. This might cause fatal errors if you do so. The slug will be automatically generated.
</Callout>

- Keywords - terms that users can search for

<Callout>
If there is more than one keyword, please use , to separate each of them. For example, apple, orange, banana
</Callout>

- Date (required) - this allows users to customise the release date shown on the website
- Feature Image - the main image of this post
- Feature Image Description - a short paragraph describing the feature image
- Body - the main content
- SEO - optional SEO setting
