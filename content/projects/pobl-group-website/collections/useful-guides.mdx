#### Fields:
- Title - the heading on the card
- Description - a short paragraph to describe the solution
- Video URL - the url of a video related to the guide or solution 
- Is Video? - toggle if you want to show the video
- Image - the feature image that will be shown on the card
- Category - related topics of the guide (see Options for Category below)
- Keywords - for the search bar in the useful guides component

<Callout>
If there is more than one keyword, please use , to separate each of them. For example, apple, orange, banana
</Callout>

Options for Category
- Repairs
- Homes
- Customer Portal
