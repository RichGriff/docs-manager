#### Fields:
- Title - determines the slug of this campaign and the title shown on SEO
- Display Title (required) - the title shown on the website
- Feature - the main image of this campaign
- Date (required) - the date of the event 
- Slug - it's used to create unique url for each campaign
- Content - the main content

<Callout>
Please don't enter / edit slug manually under any circumstances. This might cause fatal errors if you do so. The slug will be automatically generated.
</Callout>
