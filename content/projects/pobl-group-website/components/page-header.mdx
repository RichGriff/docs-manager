#### Fields:
- Heading - the section heading
- Description - A short description added to be shown under the heading
- Media -  An image shown on the right-hand side of the hero section. This field will affect the width and background colour.

> This component will take the <b>full width</b> of the page if there is no image, and take a <b>fixed width</b> (around 2/3 of the page width) if there is an image.
