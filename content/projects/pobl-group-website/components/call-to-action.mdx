Below is an example of the configuration for a call to action component. This is a large component and can be used to link to a file, show a video instead of an image also.

#### Fields:
- Heading - this the heading for the card
- Description - this will be the content inside the card
- Button - see  for more details
- Media - this is the image you want to show
- File - this is the file you want the button to be linked to
- Flip - this will show the image on the left rather than the right (default right aligned)
- Is Video - this should be selected if you intend to show a video instead of an image
- Video Url - this should be a link to the video (currently videos are added to YouTube, so you can put its url here)
- Background Colour - this allows you to change the gradient colour behind the card. Options are:
    - teal to blue (default)
    - pink to orange
    - purple to blue
