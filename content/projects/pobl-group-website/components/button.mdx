#### Fields:
- Text (required) - the text shown inside the button 
- URL (required) - the URL the button redirects
- Is External? (required) - controls the behavior when users click the button and the icon. If it's marked external, a new tab will be opened, otherwise, the browser will use the same tab to navigate.
- Colour - controls the background colour of the button
- Icon - select an icon to display or leave it blank to use the default icon
