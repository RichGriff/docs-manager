#### Fields:
- Heading - the section heading shown before the gallery
- Sub Heading - short description added to be be shown under the title
- Gallery Items - a collection of images you want to show
- Type - this changes how the gallery images are displayed on the page.
    - Slider - shows the images in a horizontal slider
    - Single - Each image fills the width of the section and shown one after another
    - Grid - Shows the images in grid based on the screen size
