> We cannot offer a form builder currently - for that reason all forms are created on request same if they require any changes.

#### CMS Configuration

The settings for the form are fairly simple and only require the user to add the component to the page and provide some values.

<img src='https://files.gitbook.com/v0/b/gitbook-x-prod.appspot.com/o/spaces%2Fqq1gvL7j0PwJkFX9MJae%2Fuploads%2FaLFh9oJTG4VxxnKskBCC%2Fform-cms.JPG?alt=media&token=0288f28e-144f-4bdd-ab09-698e1e60a561' />

#### Fields:
- Title - the section heading shown before the form
- Description - short description added to be be shown under the title
- Form - available forms to render, select which form you want to show on the page.
    - Contact Form
    - Repair Form
    - Application & Guidance - Pobl Trust Funding 
    - Compliments & Complaints
    - Report an Issue
