#### Fields:
- Title - this will be the main heading for the section on the website and will be shown before the accordion
- Accordion Items - this is where you will add each item in the accordion. Use the '+ Add Entry' link to continue adding items
- Heading - this is the question/heading for the accordion expandable section
- Content - this is a rich text editor field, allowing you to format the content the way you like it, including links, bullet lists etc.
