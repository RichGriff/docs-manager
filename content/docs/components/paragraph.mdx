#### Fields:
- Title - the main title placed before all text blocks 
- Content - the main content
- Button - a button that redirects users to certain page/external website
- Column (Required) (min. 1 / max. 2) - number of columns for the content
- Media - the image placed on the right-hand side of text blocks 

> If image (Media) exists, the paragraph will be changed back to 1 column and overwrite the setting in 'column' field.
