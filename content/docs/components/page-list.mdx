#### Fields:
- General
    - Heading - the section heading
    - Description - a short description added to be shown under the heading
- Pages
    - Title - the main title of the card
    - Decription - a short description under the card title
    - Page - select which page the card redirects users to
    - Image - the main visual image of the card
